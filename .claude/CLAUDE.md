## Structure

```
amk-content/
├── .claude/              # Project memory & config
├── articles/             # Blog posts and ideas

├── business-ideas/       # Product/business concepts
├── vibe-coding-setup/    # Claude Code configuration system (V3)

│   ├── agents/           # Specialized agent configs
```
